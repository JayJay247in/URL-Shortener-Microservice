URL Shortener Microservice

Example Usage:

POST /api/shorturl - with url in x-www-form-urlencoded body

GET /api/shorturl/ - redirects to original URL

Example Creation Output:

{
  "original_url": "https://www.freecodecamp.org",
  "short_url": 1
}

Example Redirection:

/api/shorturl/1 (will redirect if '1' exists)
